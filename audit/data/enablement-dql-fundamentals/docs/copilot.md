# CoPilot — AI-Assisted DQL

📓 [Download Exercise Notebook](https://github.com/dynatrace-wwse/enablement-dql-fundamentals/raw/main/Exercises/Workshop%20-%20Copilot%20-%20Exercises.json) | [Answer Key](https://github.com/dynatrace-wwse/enablement-dql-fundamentals/raw/main/Answers/Workshop%20-%20Copilot%20-%20Answer%20Key.json)

## Overview

Davis CoPilot can help you write DQL queries using natural language. In this exercise, you'll explore how to use CoPilot to generate, refine, and explain DQL queries.

## What you'll learn

- How to ask CoPilot to generate DQL queries from natural language
- How to refine queries through follow-up prompts
- How to use CoPilot to explain existing queries
- Best practices for prompting CoPilot for DQL

!!! tip "Davis CoPilot"
    Davis CoPilot is available in Dynatrace Notebooks. Look for the AI assistant icon to start a conversation about your data.
